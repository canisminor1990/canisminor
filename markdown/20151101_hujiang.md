# 沪江学习

沪江学习是由国内最大的在线教育平台 —— 沪江网打造的一款学习类工具应用，集移动学习和社团社交为一体。覆盖六国语言，集成听力阅读做题为一体，拥有完整的沪江社团功能体验。
